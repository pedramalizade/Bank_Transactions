# BankTransactions - Financial Request Processing System

BankRequests is a backend system designed to simulate secure and structured financial request processing using **ASP.NET Core** and Clean Architecture.

It focuses on handling banking-style operations such as request creation, validation, and processing workflows.
---
## 🚀 Overview

This project simulates a banking request workflow system where users can create financial requests that go through validation and processing pipelines.

It is designed with a strong emphasis on correctness, consistency, and maintainability.

---

## 🏗️ Architecture

Built using **Clean Architecture principles**:

- Domain Layer: Core financial business rules
- Application Layer: Request processing logic
- Infrastructure Layer: Data persistence and services
- API Layer: REST endpoints

---

## ✨ Key Features

- 💳 **Bank Request Management**
  - Create financial requests
  - Track request status
  - Validate request rules

- 🔐 **Secure Access**
  - JWT authentication
  - Role-based authorization

- ⚙️ **Request Workflow**
  - Structured processing pipeline
  - Business rule validation

- 🗄️ **Persistence Layer**
  - Entity Framework Core integration
